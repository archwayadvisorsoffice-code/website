# Archway Advisors Website

Flask-based website for Archway Advisors - Global Tax, Audit & Advisory Services.

## Features

- Multi-country service pages (India, Australia, USA, Canada, New Zealand, South Africa, UK)
- Blog system with filtering
- Contact form with email integration
- Responsive design

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Configure environment variables in `.env`:
   ```
   SECRET_KEY=your-secret-key
   MAIL_USERNAME=your-email@gmail.com
   MAIL_PASSWORD=your-app-password
   ```

3. Run the application:
   ```bash
   python app.py
   ```

## Deployment

Deployed on Vercel with custom domain: www.archwayadvisors.in
